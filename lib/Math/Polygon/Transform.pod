
=head1 NAME

Math::Polygon::Transform - Polygon transformation


=head1 INHERITANCE

 Math::Polygon::Transform
   is a Exporter


=head1 SYNOPSIS

 my @poly = ( [1,2], [2,4], [5,7], [1, 2] );

 my $area = polygon_transform resize => 3.14, @poly;


=head1 DESCRIPTION

This package contains polygon transformation algorithms.





=head1 DIAGNOSTICS

I<Error:> you need to specify 'x', 'y', 'rc', or 'line'







=head1 REFERENCES

See the Math::Polygon website at L<http://perl.overmeer.net/geo> for more details.
